# RD2.0Splash

### Installation (locally)

1. install packages:  
```sh
pip install -r requirements.txt
```

2. create local setings:  
copy and edit src/settings/local_example.py  

3. to create db:  
(sqllite db is already the repo)  
```sh
bash init.sh
```

PS.  
(reserved) npm install
